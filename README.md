# Rutter (rutter)
Rutter is a unified RESTful API for B2B financial products that connects to commerce, payments, accounting, and ads platforms through a single API. Rutter enables developers to read, update, write, and remove data across 60+ platforms including accounting, commerce, payments, and advertising systems used globally by business customers.
